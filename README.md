# 프로젝트 샘플
### API를 사용하여 영화리스트를 받아오고 링크를 연결하여 영화에 대한 설명을 볼 수 있는 애플리케이션입니다.
![movieApp_sample](https://user-images.githubusercontent.com/73148498/166879363-858147a3-29f4-4c14-893f-f6da70011a96.PNG)


# 주요기능
* React 
  * create-react-app을 사용하여 기본 리액트애플리케이션 베이스로 사용하였습니다.
  * useState / useEffect 를 사용하였습니다.
  * PropType을 사용하여 Typescript처럼 타입을 정의해 보았습니다.
